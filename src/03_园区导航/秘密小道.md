# 园区导航

## 🔑 三条秘密小道（谨记！）

### 🟠 小道1：门口 → 哈利波特区
- 从门口到哈利波特区，一定要从 **小黄人萌乐园** 穿过去
- 比绕小黄人神偷奶爸闹翻天 **少走至少800米**

### 🟠 小道2：大门口 → 侏罗纪区
- 一定不要绕到小黄人区再去
- 一定要 **奔着假山走**，从假山底下穿过去
- 穿过去就是侏罗纪区的两个项目

### 🟠 小道3：侏罗纪大冒险 → 水世界（节假日限定）
- 人多的节假日在侏罗纪大冒险，**右边会开通一个通道**
- 直达水世界
- 不用再从变形金刚区或者哈利·波特区绕过去
